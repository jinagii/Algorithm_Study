공개키 암호화 알고리즘 (RSA)

- 특성
공개키-암호키가 한쌍을 이룸
공개키로 안호화한 내용은 개인키로만, 개인키로 암호화한 내용은 공개키로만 해독가능 
큰 숫자는 소인수분해하기 힘들다는 특징을 이용

- 키 생성 과정
1. 소수 p, q 준비 N = p*q
2. (p-1) * (q-1) = phi(N) 
3. e < phi(N) , 서로소 임
4. (d * e) / phi(N) 때 나머지 1인 d를 구함

# 공개키 = < N, e > // 개인키 = < N, d >
# A는 공개키를 B에게만 공개 - B는 공개키로 메세지 암호화

- 암호화 
1. B->A 로 M이란 메세지 발신 시, M -> m (m<N) 변환 
2. c=m^e mod N -> c를 A에게 전송

- 복호화
1. m=c^d mod N 
2. A는 암호화 메세지 c를 받음 / N, d를 알고있음
3. A는 m으로 M을 찾아낼 수 있음
